# Task Manager API

Simple Task Manager built with Flask, JWT authentication and SQLite.

## Features

- User registration and login
- JWT authentication
- Create tasks
- List tasks
- Update tasks (title / done)
- Delete tasks
- Simple web interface

---

## How to run the project

### 1. Clone the repository

git clone https://github.com/Berganty-Sec/task-manager-api.git
cd task-manager-api


---

### 2. Create virtual environment

python -m venv venv


---

### 3. Activate environment

Windows:

venv\Scripts\activate

---

### 4. Install dependencies

pip install -r requirements.txt


---

### 5. Run the application

python run.py


Open in browser:

http://127.0.0.1:5000
